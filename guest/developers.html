Web Services

UuidRegistry.Com offers a simple web service, that allows to create and register as many Universally Unique Identifier (UUID) string(s) as needed.

Register an UUID

        http://uuidregistry.com/api/?key=YOUR_KEY&description=UUID_DESCRIPTION

Tools

Human-friendly UUID View in Action

A common complain with standard UUIDs are that they are unreadable. This no longer has to be the case. This small web service will generate a human-friendly UUID, which is easily readable and perfectly valid. It will generate a UUID looking like: 20141219-1003-1033-473572-808278214600. Benefits of using: easily readable; by looking at it provides date and time information when was generated; can be ordered by date of creation; UUID valid. You may also choose to store the value in the database using the provided bases -- Base10, Base16, Base32, Base36, Base62, Base64, Base75 -- thus saving extra space. Available in JSON and JSONP. 

        http://uuidregistry.com/tools/index.php?cmd=human-uuid

Human-friendly Unique ID View in Action

A human-friendly UUID is too long. Here is a unique ID generator, which is not so long, yet it provides unique string. The unique ID will look like: 20141219-100951-489936. Benefits of using: easily readable; by looking at it provides date and time information when was generated, can be ordered by date of creation. You may also choose to store the value in the database using the provided bases -- Base10, Base16, Base32, Base36, Base62, Base64, Base75 -- thus saving extra space. Available in JSON and JSONP. 

        http://uuidregistry.com/tools/index.php?cmd=human-uid

Timestamp-based Unique ID View in Action

All previous versions are too long? Use the UNIX timestamp based unique ID generator. The unique ID will look like: 1418983968. Benefits of using: easily readable; can be ordered by date of creation. Available in JSON and JSONP. 

        http://uuidregistry.com/tools/index.php?cmd=timestamp-uid
